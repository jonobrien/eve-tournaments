# eve_js

Displaying public CREST data grouped by endpoint

```bash
$ npm install
$ node server.js
```

# endpoints
- /tournaments/
- /tournaments/teams/`<id>`/
- /tournaments/`<id>`/series/
- /tournaments/`<idX>`/series/`<idY>`/matches/
